# 快速开始指南

## 1. 安装依赖（首次使用）

```bash
# 安装ROS2包
sudo apt install -y \
    ros-humble-slam-toolbox \
    ros-humble-nav2-map-server \
    python3-serial

# 设置串口权限
sudo usermod -aG dialout $USER
# 重新登录后生效
```

## 2. 编译项目

```bash
cd ~/chapt7_pro_ws
colcon build --packages-select czybot_navigation2
source install/setup.bash
```

## 3. 测试STM32通信（推荐）

```bash
# 先测试STM32通信是否正常
python3 src/czybot_navigation2/scripts/test_stm32_comm.py /dev/ttyUSB0

# 如果能收到里程计数据，说明通信正常
```

## 4. 启动SLAM建图

### 方式一：使用脚本（推荐）

```bash
bash src/czybot_navigation2/scripts/start_slam.sh
```

### 方式二：直接启动

```bash
source install/setup.bash

ros2 launch czybot_navigation2 slam_real_robot.launch.py \
    lidar_port:=/dev/ttyCH343USB0 \
    stm32_port:=/dev/ttyUSB0
```

## 5. 键盘控制小车

在新终端中：

```bash
source install/setup.bash
ros2 run czybot_navigation2 teleop_key
```

按键说明：
- `w` - 前进
- `s` - 后退  
- `a` - 左转
- `d` - 右转
- `空格` - 停止
- `q` - 退出

## 6. 保存地图

建图完成后：

```bash
ros2 run nav2_map_server map_saver_cli -f ~/my_map
```

## 常见问题

### 串口权限错误

```bash
sudo chmod 666 /dev/ttyUSB0
sudo chmod 666 /dev/ttyCH343USB0
```

### 找不到串口设备

```bash
# 查看所有串口
ls -l /dev/tty*

# 常见设备名：
# LD14P: /dev/ttyCH343USB0 或 /dev/ttyCH343USB1
# STM32: /dev/ttyUSB0 或 /dev/ttyACM0
```

### 收不到里程计数据

1. 检查STM32程序是否正确烧录
2. 使用测试脚本验证通信
3. 检查串口波特率（应为115200）
